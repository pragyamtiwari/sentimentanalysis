Sentiment Analysis

This sentiment analysis website, built using TextBlob, analyzes the emotions behind the input text and provides a score from -100 (very negative, like “This is awful!”) to 100 (very positive, like “I love this!”). Sentiment analysis can have uses in bias spotting, stocks research, and artificial customer service.
